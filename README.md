# electron-printer

A electron based priting demo app.

## GTD

1. Menu System
2. Order tracking system/dashboard

## Tech Stack

- Electron
- Next.js
- NextUI
- node

## Get Started

### Preperation

1. make sure you have printer setup
2. install all pacakges, suggest use nvm to mange the node tech stacks

### frontend

```
cd frontend
bun install 
bun dev
```

### backend

in new terminal

```
bun install
bun start
```
